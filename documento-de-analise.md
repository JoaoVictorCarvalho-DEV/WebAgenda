## Algoritmos de busca:

### Busca linear
Complexidade:
- Pior caso: O(n)
- Caso médio: O(n)
- Melhor caso: O(1)

### Busca binária
Complexidade:
- Pior caso: O(log n)
- Caso médio: O(log n)
- Melhor caso: O(1)

>Foi utilizada a iteração para as funções de busca

## Algoritmo de ordenacao

### Quick sort
Complexidade:
- Pior caso: O(n²)
- Caso médio: O(n log n)
- Melhor caso: O(n log n)

**Implementação:**

Foi utilizada recursão para a ordenação


>A maior parte das estruturas de dados utilizadas no projeto foram array de objetos: utilizado para armazenar e manipular listas de grupos e contatos no front-end.
